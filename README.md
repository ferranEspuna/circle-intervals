# Circle Intervals

Interactive React visualizer for interval sets, sumsets, and dilations on the
circle group.

## Development

```bash
npm install
npm run dev
```

## Deploy

Pushing to `main` runs the GitHub Actions workflow in
`.github/workflows/deploy-pages.yml`, builds the Vite app, and deploys it to
GitHub Pages.

Expected Pages URL:

https://ferranespuna.github.io/circle-intervals/
